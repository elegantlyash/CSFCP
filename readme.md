# Portfolio Website – CSF Mini Project

This is a simple, clean, and responsive **Portfolio Website** created as part of the *Computer Science Fundamentals (CSF) Mini Project*. It showcases personal information, projects, and basic web development skills using **HTML** and **CSS**.
The project also demonstrates the use of **Git**, **GitHub**, proper documentation, commit history, screenshots, and recommended project structure.

## Features

* Clean and professional landing page
* “About Me” section
* “My Projects” section
* Simple and responsive layout
* Developed using pure HTML & CSS
* Organized folder structure
* Includes documentation and screenshots
* Version-controlled using Git

## Project Structure

## Technologies Used

* **HTML5**

## How to Run the Project

1. Download or clone the repository:

   ```
   bash
   git clone https://github.com/SNEHANSH1306/portfolio-mini-project.git
   ```
2. Open the project folder.
3. Double-click `index.html`
   → The website will open in your browser.

No extra installations are required.
